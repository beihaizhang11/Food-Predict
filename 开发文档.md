# 🎓 毕业设计开发文档（完整最终版）

## 1. 项目概述

**项目名称**
餐饮点评趋势预测与知识图谱可视化系统

**项目背景与意义**
随着线上餐饮点评（如美团外卖点评）数据日益丰富，如何从用户文本中挖掘有价值的信息用于行业分析与未来趋势预测成为重要研究方向。本系统旨在利用 NLP、知识图谱与 LSTM 时间序列预测技术构建一个完整的数据分析与可视化平台，让用户直观了解餐饮评价动态及未来趋势。

**核心目标**

* 自动提取点评文本中的情感和实体信息
* 构建餐饮领域知识图谱
* 使用 LSTM 模型进行趋势预测
* 前端展示完整工作流 + 趋势图 + 图谱交互

## 5. Mock 数据策略

提供至少 **1000+ 条模拟点评数据** 用于训练与展示效果。数据字段包括时间、用户、菜品、情感及标签。

你可以使用如下 Python 脚本生成 JSON 数据：

```python
import random, datetime, json

dishes = ["宫保鸡丁","牛肉面","披萨","炸鸡汉堡","寿司"]
tags_pool = ["好吃","太咸","服务好","环境差","等太久","价格实惠"]

data = []
for i in range(1000):
    t = datetime.datetime(2023, random.randint(1,12), random.randint(1,28),
                          random.randint(10,22), random.randint(0,59), random.randint(0,59))
    dish = random.choice(dishes)
    rating = round(random.uniform(2.5, 5.0),1)
    tags = random.sample(tags_pool, random.randint(1,3))
    text = f"在这家店吃了{dish}，感觉" + "，".join(tags) + "。"
    data.append({"id":i+1,"user_id":f"U{random.randint(1000,9999)}","shop_id":f"S{random.randint(100,999)}",
                 "dish":dish,"rating":rating,"review_text":text,"review_time":str(t),"tags":tags})

with open("mock_reviews.json","w",encoding="utf-8") as f:
    json.dump(data,f,ensure_ascii=False)
```

## 7. 核心算法与模型

### 7.1 文本分析与 NLP

* 使用 BERT/SBERT 提取语义向量，并进行情感分析与命名实体识别
* 实体识别并构建节点/属性，如菜品、价格、服务等

### 7.2 LSTM 趋势预测

基于时间序列（如按周、月的平均评分和情感指标）训练 LSTM 模型预测未来趋势。

```python
import torch.nn as nn

class TrendLSTM(nn.Module):
    def __init__(self,input_dim,hidden_dim):
        super().__init__()
        self.lstm = nn.LSTM(input_dim,hidden_dim,batch_first=True)
        self.fc = nn.Linear(hidden_dim,1)
    def forward(self,x):
        out,_ = self.lstm(x)
        return self.fc(out[:,-1,:])
```

模型训练流程包括数据标准化、划分训练/验证集、训练与保存模型结果。

## 9. 工作流可视化

### 9.1 工作流定义与数据结构

为可视化系统执行逻辑，自定义工作流节点与连线 JSON：

```json
{
  "nodes":[
    {"id":"data-import","label":"数据采集"},
    {"id":"clean","label":"数据清洗"},
    {"id":"nlp","label":"文本分析"},
    {"id":"kg","label":"图谱构建"},
    {"id":"predict","label":"趋势预测"}
  ],
  "edges":[
    {"from":"data-import","to":"clean"},
    {"from":"clean","to":"nlp"},
    {"from":"nlp","to":"kg"},
    {"from":"kg","to":"predict"}
  ]
}
```

该数据可由后端直接返回以供前端渲染。
